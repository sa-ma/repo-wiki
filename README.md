# RepoWiki

Generate AI-powered documentation for any public GitHub repository. Paste a repo URL, get a structured wiki with features, code snippets, and an interactive Q&A chat.

## Setup

```bash
npm install
```

Create a `.env.local` file:

```
GITHUB_TOKEN=your_github_token
OPENAI_API_KEY=your_openai_api_key
```

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
